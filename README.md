# DOJI Gold Price

An automated tool that fetches and displays gold price data from DOJI (Doji Gold Trading Platform).

## 🌐 Live Demo

Check out the live demo here: [DOJI Gold Price Tracker](https://doji-gold.netlify.app/)

## 📋 Features

- 🔄 Automatic gold price updates
- 📊 Real-time price tracking from DOJI
- 📱 Responsive web interface
- 🎨 Clean and user-friendly UI

## 🛠️ Technologies

- **Frontend**: HTML, CSS, JavaScript
- **Deployment**: Netlify

## 📦 Project Structure

- `HTML` - 90.6%
- `Shell` - 5%
- `JavaScript` - 4.4%

## 🚀 Getting Started

### Prerequisites

- Node.js (optional, depending on build setup)
- Git

### Installation

```bash
git clone https://github.com/dunglex/doji-gold-price.git
cd doji-gold-price
```

### Running Locally

Simply open `index.html` in your web browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Or using Node.js
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
